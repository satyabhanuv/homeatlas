# HomeAtlas — Domain Name Options

**Important caveat up front:** I checked availability via web search, not direct WHOIS. A name with no search results probably isn't actively in use, but it might be parked silently. **Before buying, paste your shortlist into [Cloudflare Registrar](https://dash.cloudflare.com/?to=/:account/domains/register) or [Namecheap](https://www.namecheap.com/domains/) for the definitive answer.** Cloudflare's prices are at-cost (~$10/yr for `.com`), Namecheap adds a small markup but the search UX is best.

The `.com` space is essentially "all owned by someone" — even names with no website are usually held by domain investors. Realistic options are: pick a less-obvious `.com` name, pay $100-5000 for a parked premium `.com`, or use a newer TLD like `.app`/`.io`/`.co`.

---

## How to read the table

| Status | Meaning |
|---|---|
| 🚫 **Taken** | Confirmed active website on the domain |
| 💰 **For sale** | Domain is held by an investor/broker, listed on a marketplace (typically $500-$5,000+) |
| 🤔 **Possibly available** | No active site found in search; needs registrar verification |
| ❓ **Unverified** | Couldn't determine — check directly |

## My specific recommendation

If I were buying tonight, this is the order I'd try (and the price expectation):

1. **`homeatlas.app`** — ~$15/yr at Cloudflare. Best balance of brand continuity and availability. **This is what I'd buy.**
2. **`homedossier.com`** — if available at registrar (~$10/yr). Distinctive, ownable, sets up the "comprehensive home file" narrative.
3. **`knowmyhome.com`** — if available (~$10/yr). Clearest value proposition.
4. **`homeatlas.io`** — ~$50/yr at Cloudflare (`.io` is more expensive). Good fallback.
5. **`homecompass.com`** — buy from Atom if budget allows ($500-$2500 likely). Most premium/memorable but pricey.

**My single pick if you want one: `homeatlas.app`.** It's modern, on-brand with everything we've built, almost certainly available, and `.app` automatically requires HTTPS which is a nice security signal.

## Action checklist

Before you buy:

- [ ] Pick your top 3-5 from this list
- [ ] Verify availability at https://dash.cloudflare.com/?to=/:account/domains/register (cheapest registrar, prices are at-cost)
- [ ] Check trademark conflicts at https://tmsearch.uspto.gov for names you're seriously considering
- [ ] Search "[name] app" on Apple App Store + Google Play to ensure no existing app of the same name in our space
- [ ] Search Instagram + TikTok handles to see if `@homeatlas` style handle is available (matters for marketing later)
- [ ] Buy the `.com` AND `.app` together if both are available — costs ~$25/yr and prevents impersonation
